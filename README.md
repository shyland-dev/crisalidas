# [crisalidas](https://shyland-dev.github.io/crisalidas/)
Artesanato em PVC: Luminárias, Brindes, Troféus, Placas Decorativas e Chaveiros.

---
## Prerequisites:
- [NodeJS](https://nodejs.org/)

## How to install:
1. Install packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    npm start
    ```
3. Build project
    ```bash
    npm run build
    ```
4. Deploy project (gh-pages)
    ```bash
    npm run deploy
    ```
